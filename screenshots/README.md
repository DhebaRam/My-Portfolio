# Project Screenshots

Place your project screenshots in this directory.

## Naming Convention

Name your screenshots according to the project name:
- `nawedey-1.jpg`, `nawedey-2.jpg`, etc.
- `renew360-1.jpg`, `renew360-2.jpg`, etc.
- `figk12-1.jpg`
- `eshealthcare-1.jpg`, `eshealthcare-2.jpg`, etc.
- `eswasthya-1.jpg`
- `scoremore-1.jpg`, `scoremore-2.jpg`, etc.
- `starstudent-1.jpg`
- `cheese-1.jpg`, `cheese-2.jpg`, etc.
- `defalcor-1.jpg`
- `loyalty-1.jpg`
- `resume-1.jpg`
- `avapp-1.jpg`

## Image Requirements

- Recommended format: JPG or PNG
- Recommended aspect ratio: 9:16 (portrait) for mobile app screenshots
- Recommended size: 1080x1920px or similar
- Keep file sizes optimized for web (under 500KB per image)

## How to Add Screenshots

1. Take screenshots of your apps
2. Name them according to the convention above
3. Place them in this `/public/screenshots/` directory
4. The portfolio will automatically display them in the project cards
